# Terminal Grok

This program works in linux, use your api key to connect to openrouter.ai
Grok will say hello to you, and help you finish daily bash jobs

1. Edit start.bat
    set "workspace=E:\_Workspace\TerminalGrok" , or any where you like, it's a mirror python project

2. double click start.bat
    it will create a workspace, configure python venv(py.exe) required, install packages
    it will send grok.bat to your \...\python312\ path, and you can use >grok hello in cmd to chat with it
    it will start main.py and run grok agent

3. open cmd any where
    type >grok and check grok's hello message
    type >grok hello
    and wait grok's response
    type >grok /b the bash command decription you want it to do
    and wait grok's action
